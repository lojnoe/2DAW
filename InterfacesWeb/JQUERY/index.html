Ejercicios JQUERY

  1. Una caja de texto donde el usuario introduce la información.
  2. Botón de añadir título H1 (Esta opción debe desaparecer al añadir el título, es decir, solo puede haber un H1).
  3. Botón cambiar H1 (Este botón solo debe aparecer si se ha introducido un H1).
  4. Botón de añadir título H2.
  5. Botón de añadir título H3.
  6. Botón de añadir párrafo.
  7. Botón para añadir imagen desde url.
  8. Botón para cambiar la fuente de texto de todo el artículo.
  9. Botón para cambiar el color de fondo del artículo.
  10. Botón para cambiar el color de todo el texto del artículo.
  11. Botón "deshacer" que elimina el último elemento introducido en el artículo.
  12. Botón para eliminar el artículo y empezar de 0.
  13. Botón para alternar "modo oscuro" (fondo negro y texto blanco).
  14. Botón para alternar entre mostrar imágenes o no [ Las imágenes deben desvanecerse ("fade") ].
